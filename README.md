# Yandex Afisha Automated Tests

Автоматизированные тесты для сайта https://afisha.yandex.ru с использованием Playwright + TypeScript.

## Требования

- Node.js 18+
- Docker (для запуска в контейнере)
- Allure Commandline (для генерации отчетов)

## Установка

```bash
npm install
npx playwright install chromium
